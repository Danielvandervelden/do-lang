  <name>Task 1: Create package infrastructure</name>
  <files>package.json, .gitignore, bin/install.cjs</files>
  <action>
Create the core package files:

1. **package.json** - per D-01 (package name `do-lang`) and D-02 (version 0.1.0):
```json
{
  "name": "do-lang",
  "version": "0.1.0",
  "description": "Token-efficient meta programming language for Claude Code and Codex",
  "files": ["skills", "bin"],
  "scripts": {
    "postinstall": "node bin/install.cjs"
  },
  "keywords": ["claude-code", "skills", "ai", "codex", "meta-programming"],
  "license": "MIT",
  "engines": {
    "node": ">=16.7.0"
  },
  "repository": {
    "type": "git",
    "url": "git+https://github.com/globalroo/do.git"
  }
}
```

2. **.gitignore** - standard Node.js patterns:
```
node_modules/
*.tgz
.yalc/
yalc.lock
.DS_Store
```

3. **bin/install.cjs** - postinstall script per research patterns:
- Use `require()` syntax (CommonJS for .cjs)
- Source: `path.join(__dirname, '..', 'skills', 'do')`
- Target: `path.join(os.homedir(), '.claude', 'skills', 'do')`
- Create parent directories with `fs.mkdirSync({ recursive: true })`
- Copy with `fs.cpSync({ recursive: true })`
- Log success message
- Handle case where source doesn't exist gracefully (for dev installs)
  </action>
  <verify>
    <automated>node -c bin/install.cjs && cat package.json | node -e "const p=require('./package.json'); if(p.name!=='do-lang'||p.version!=='0.1.0') process.exit(1)"</automated>
  </verify>
  <done>package.json has correct name/version, bin/install.cjs is valid Node.js, .gitignore exists</done>
</task>
